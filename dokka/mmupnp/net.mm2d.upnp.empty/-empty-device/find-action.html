findAction

fun findAction(name: String): Action?

Overrides Device.findAction

Find Action by Action name.

Search from all services and return the first one found. If it can not be found, it returns null.

If you want to obtain the action of a specific service, use Service.findAction after obtaining Service using findServiceById or findServiceByType.

Parameters

name - Action name

Return
Action
