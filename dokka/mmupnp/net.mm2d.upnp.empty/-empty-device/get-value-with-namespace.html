fun getValueWithNamespace(namespace: String, name: String): String?

Overrides Device.getValueWithNamespace

Get the value of the tag described in the Description.

Non-standard tags can also be obtained, but no method for obtaining attribute values is provided. when the same tag is described more than once, the value described at the end is returned. A namespace is not a prefix but a URI. If the value does not exist, null is returned.
