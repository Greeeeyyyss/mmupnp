DUAL_STACK

DUAL_STACK

Use dual stack of IPv4 / IPv6.

Prioritizes IPv4 over IPv6, but prefers IPv6 if the IPv4 address is a link local address (assigned by APIPA).

Only link local addresses are used as an IPv6 address system. It does not support multicast on site local addresses.
