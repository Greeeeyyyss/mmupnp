isKeepAlive

var isKeepAlive: Boolean

keep-alive

Default is true. If true, the connection is continued if the response can be keep-alive. Even if true, the connection is disconnected if the response is not capable of keep-alive. If false, the connection is disconnected regardless of the response.

Also, even if true or false, the contents of HttpRequest passed in the post argument will not be changed.

If you need to describe keep-alive in the header, you need to set it in HttpRequest before calling post.
