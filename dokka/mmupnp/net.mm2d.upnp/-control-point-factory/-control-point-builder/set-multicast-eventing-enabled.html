setMulticastEventingEnabled

fun setMulticastEventingEnabled(enabled: Boolean): ControlPointFactory.ControlPointBuilder

Set whether to use multicast eventing.

Multicast eventing is an experimental feature. Because no other implementation has been found and compatibility has not been verified at all.

Default is false. If set to true, multicast events can be received.

Parameters

enabled - true, enable multicast eventing. false, otherwise
