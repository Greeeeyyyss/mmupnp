fun setCallbackHandler(handler: (Runnable) -> Boolean): ControlPointFactory.ControlPointBuilder

Set callback handler.

This is used as a TaskExecutor. All callbacks are executed from the callback thread. This argument is used to specify the callback thread. If not specified, single Thread Executor is used.
