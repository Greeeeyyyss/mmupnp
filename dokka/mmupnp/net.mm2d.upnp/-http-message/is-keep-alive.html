abstract fun isKeepAlive(): Boolean

Returns whether this is Keep-Alive from the header value

In case of HTTP/1.0, "Connection" is "keep-alive" In case of HTTP/1.1, "Connection" is not "close" Judge as KeepAlive and return true.

true if KeepAlive
