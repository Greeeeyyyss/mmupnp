getNetworkInterfaceList

@JvmStatic fun getNetworkInterfaceList(): List<NetworkInterface>

Return a list of all network interfaces in the system.

The return value of java.net.NetworkInterface.getNetworkInterfaces is changed to java.util.List instead of java.util.Enumeration. If there is no interface or if java.net.SocketException occurs, an empty List is returned.

Return
all network interfaces in the system.
